# Space Defender – PyGame

Proyecto desarrollado en Python utilizando la librería **PyGame** como trabajo práctico académico.

## Descripción
Space Defender es un juego 2D en el que el jugador controla una nave espacial.
El objetivo es esquivar a los enemigos que aparecen de forma aleatoria desde
la parte superior de la pantalla y sobrevivir el mayor tiempo posible.

## Funcionalidades
- Menú principal
- Control del jugador mediante el teclado
- Enemigos generados aleatoriamente
- Sistema de vidas
- Puntaje
- Funciones personalizadas

## Tecnologías utilizadas
- Python 3
- PyGame

## Ejecución del proyecto
Este proyecto fue desarrollado para ejecutarse de forma local, ya que PyGame
no funciona en navegadores web.

Para ejecutarlo:

1. Instalar PyGame:
   pip install pygame

2. Ejecutar el archivo principal:
   python main.py
